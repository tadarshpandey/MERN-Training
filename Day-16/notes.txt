>> Hooks: React Hooks are built-in JavaScript functions that allow you to use state and 
          other React framework features inside functional components.

>> Prop drilling: Prop drilling (also known as "threading props") is the process of passing
                  data down through multiple nested components in React,
                  even when the intermediate components don't need the data themselves

- useState – Manages and updates component state in React.
[Component state is the data that belongs to a React component and can change over time.

When state changes, React automatically re-renders the component to show the updated data.]

- useEffect – Performs side effects like API calls, timers, or DOM updates.


- useContext – Accesses and shares data across components without prop drilling.

- useRef – Stores mutable values and directly references DOM elements without re-rendering.

- useReducer – Manages complex state logic using actions and a reducer function.

- useCallback – Memoizes a function to prevent unnecessary re-creation on re-renders.

- useMemo – Memoizes a computed value to optimize expensive calculations.
